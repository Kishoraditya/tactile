# Manual Audio Recording Guide (Marathi)

This guide lists all the Marathi audio files required for the ToothBuddy application. 
Please record the **Dialogue** and save the file with the exact **Filename** inside the specified **Folder**.

All files should be **MP3 format**.

**Root Directory:** `frontend/public/audio/mr/`

---
